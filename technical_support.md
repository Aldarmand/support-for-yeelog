# Technical Document for Yeelog (Voice Accounting App)


## 1. Product Overview

This is a mobile application focused on "convenient accounting", with the Chinese name "小语记 (Xiaoyu Ji)" and the English name "yeelog". Its core positioning is to help users efficiently manage personal and family finances through multi-dimensional recording methods and practical tools.

## 2. Core Function Modules

### 2.1 Bill Recording Functions



* Voice Recording: Allows users to automatically generate bills via voice input without manual entry;

* Handwriting Recording: Provides a handwriting input mode to meet the needs of users who prefer handwriting for accounting;

* Bill Management: Supports individual deletion of single bills and batch deletion function for quickly cleaning up redundant data.

### 2.2 Practical Tool Module



* Financial Knowledge Section: Integrates basic financial science content to help users improve financial literacy;

* Currency Exchange Tool: Updates exchange rate data in real time and supports quick conversion between multiple currencies;

* Bill Reminder: Allows setting custom reminder times to avoid missing bills to be recorded or payment deadlines;

* Shared Ledger: Supports multiple users to jointly manage the same ledger, suitable for scenarios such as families and co-rentals;

* Scan-to-Record: Automatically identifies bill information and generates records by scanning the QR code on consumption vouchers.

### 2.3 Data Processing Functions



* Data Analysis: Conducts multi-dimensional analysis of user income and expenditure data (e.g., proportion of classified expenditures, monthly income and expenditure trends);

* Data Statistics: Automatically generates visual statistical reports (e.g., pie charts, line charts) to intuitively present financial status.

## 3. Membership Subscription Service

The product offers a membership subscription model with two packages:



* Monthly Membership: Charged by calendar month, unlocks all premium features after activation;

* Annual Membership: Charged by calendar year, with higher cost-effectiveness, and also enjoys full access to premium features after activation.
